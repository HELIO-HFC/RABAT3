Ce dossier contient les routines permettant de selectionner manuellement des types III sur des spectres.
Ces types III peuvent ensuite être utilisés pour optimiser le taux de detection du code RABAT3.

Les scripts *.batch.pro peuvent être executés sous idl pour lancer la selection.
Il y a un script par jeu de données.

L'execution requiert SolarSoft, ainsi que l'existence de plusieurs variables d'environnements.
Pour charger ces variables, on pourra utiliser un des scripts du sous-dossier /rabat3/setup.
